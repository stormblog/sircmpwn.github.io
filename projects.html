Projects

I have lots of different open source projects. Only the bigger ones are listed here, but there are lots more on my GitHub account.

Minecraft

I like to reverse engineer Minecraft and reimplement it in .NET.

  • Craft.Net is a bunch of .NET libraries (including a custom server, client, Anvil editng, etc) for working with Minecraft.
  • PartyCraft is a third-party Minecraft server based on Craft.Net.
  • SMProxy is a proxy for Minecraft network protocol inspection and reverse engineering.

KnightOS

KnightOS is a thid-party operating system for Texas Instruments calculators, and there are lots of projects related to it.

  • kernel is written from scratch in z80 assembly.
  • KnightOS is all the userspace, also in z80 assembly.
  • sass is the assembler I wrote (since most z80 assemblers are pretty dated), it's done with C#.
  • KDoc is a system that generates HTML documentation based on comments in z80 assembly code.
  • knightsoft.org is the website I made for everything.

Miscellaneous

  • RedditSharp is a .NET library for talking to Reddit.
  • bf-irc-bot is an IRC bot written in Brainfuck.
  • ChatSharp is a .NET IRC library that's a lot nicer than bf-irc-bot.
  • Patchy is a bittorrent client for Windows.
  • WebSharp is an experimental web framework for C# that tries to emulate node.js-style development.
